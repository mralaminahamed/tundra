## What Tundra manages

<CardGrid>
  <Card title="Multi-runtime sites" icon="rocket">
    PHP, Node.js, Python, Go, Rust, Ruby, .NET — blue/green deployments, atomic rollback, 13 starter templates.
  </Card>
  <Card title="WordPress" icon="puzzle">
    WP-CLI provisioning, per-install DB isolation, plugin/theme management, staging and cloning.
  </Card>
  <Card title="Multi-server fleet" icon="laptop">
    SSH-based agent install, mTLS gRPC mesh, cross-server site move, maintenance windows.
  </Card>
  <Card title="Databases" icon="setting">
    PostgreSQL, MySQL, MariaDB, Valkey — CRUD, access control, connection strings.
  </Card>
  <Card title="DNS & Mail" icon="email">
    PowerDNS zones, 30 provider-ready DNS templates, Postfix + Dovecot + Rspamd + DKIM.
  </Card>
  <Card title="Backups" icon="document">
    restic + GPG to S3, B2, R2, Wasabi, SFTP, or local. Preview-then-confirm restore.
  </Card>
  <Card title="Plugin system" icon="open-book">
    Wasmtime sandbox, WIT SDK, capability gate. Official plugins: WordPress, GitHub, Namecheap, MCP.
  </Card>
  <Card title="MCP AI integration" icon="star">
    Connect Claude Code, Claude Desktop, Cursor, Zed. Manage infrastructure from your AI editor.
  </Card>
</CardGrid>
