## Quick install

```bash
curl -fsSL https://raw.githubusercontent.com/mralaminahamed/tundra/main/installer/install.sh | sudo bash
```

Runs on Ubuntu 24.04, Debian 12/13, RHEL 9/10. After install, visit the printed setup URL to create your owner account.
